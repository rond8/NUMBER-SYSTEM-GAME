Conversion Guide

1. Decimal to Any Base (Division Method)

Divide the decimal number by the target base. Keep the remainder. Repeat until the quotient is 0. Read remainders bottom-up.

Example: 13 to Binary (Base 2)
13 ÷ 2 = 6 (Rem: 1)
6 ÷ 2 = 3 (Rem: 0)
3 ÷ 2 = 1 (Rem: 1)
1 ÷ 2 = 0 (Rem: 1) → 1101
2. Any Base to Decimal (Positional)

Multiply each digit by the base raised to its position power ($base^{position}$), then sum them up.

Example: $101_2$ to Decimal
$(1 \times 2^2) + (0 \times 2^1) + (1 \times 2^0)$
$4 + 0 + 1 = 5$
